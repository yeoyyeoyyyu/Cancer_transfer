# 유방암의 임파선 전이 예측 AI 경진대회

## 개요
림프절(임파선)은 암의 전이에 매우 치명적인 역할을 하기 때문에 림프절 전이 여부와 전이 단계를 파악하는 것이 암을 치료하고 진단하는 것에 있어서 매우 중요
<br>

## 주제
유방암 병리 슬라이드 영상과 임상 항목을 조합하여 유방암의 임파선 전이 여부 이진 분류
<br>
